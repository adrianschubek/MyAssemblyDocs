# #include
Includes a file.
!!!info 
This should be the top-most statement.
!!!
### Syntax
```
#include <filename>
```
##### Parameters
- *filename* [!badge variant="danger" text="required"] Type: [!badge variant="info" text="Name"](/api-docs/datatypes/#name)
### Example
```
#include std.lib
```
