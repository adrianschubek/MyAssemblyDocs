# pop
Pops a value from the stack.
### Syntax
```
pop <register>
```
##### Parameters
- *register* [!badge variant="danger" text="required"] Type: [!badge variant="info" text="Register"](/api-docs/standard/datatypes.md#register) 
### Example
```
pop r0
```
